This project is a modular Java-based two-player strategy game built on graph data structures. It supports both adjacency list and adjacency matrix representations and integrates a Divide-and-Conquer AI for optimal move selection.
The system demonstrates strong application of graph theory, recursive strategy computation, and object-oriented design principles.
Core Features
• Graph abstraction via interface
• Adjacency List implementation
• Adjacency Matrix implementation
• Config-driven graph selection
• Divide-and-Conquer AI opponent
• Interactive GUI
• Clean separation of logic, AI, and UI
Technical Concepts
• Graph Data Structures
• Recursive Strategy Evaluation
• Deterministic Game Modeling
• Object-Oriented Architecture
• Interface-based Design
• Time Complexity Analysis
Architecture
Graph Layer → GameLogic → AI Engine → GUI
How to Run
Compile all Java files and execute Main.java.
Strength
Demonstrates structured problem-solving, modular system design, and AI-based decision modeling.
